# mica-metrics 组件

## 功能
- druid 指标收集。
- ~~undertow 指标收集~~（spring boot 4 不支持）。

## 使用
### maven
```xml
<dependency>
  <groupId>net.dreamlu</groupId>
  <artifactId>mica-metrics</artifactId>
  <version>${version}</version>
</dependency>
```

### gradle
```groovy
compile("net.dreamlu:mica-metrics:${version}")
```
